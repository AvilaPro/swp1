Para ingresar recuerde colocar como sigue:

localhost/desafio3_avila?apikey="Your apikey"

Para hacer consultas fitradas por id:

localhost/desafio3_avila?apikey="Your apikey"&id="number"

Para hacer consultas fitradas por nombre:

localhost/desafio3_avila?apikey="Your apikey"&name="nombre"

Para hacer consultas fitradas por apellido:

localhost/desafio3_avila?apikey="Your apikey"&lastname="apellido"

Para hacer consultas fitradas por el destino:

localhost/desafio3_avila?apikey="Your apikey"&destination="destino"
